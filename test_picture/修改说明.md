## 主要修改内容

### 1. school.php 的修改

**修改前：**
```php
<div class="card-image">
    <img src="get_image.php?id=<?php echo $row['sid'] ?>" class="card-image">
</div>
```

**修改后：**
```php
<div class="card-image">
    <img src="get_image.php?id=<?php echo $row['sid'] ?>" alt="<?php echo $school_name; ?>">
</div>
```

**关键点：**
- 删除了 `img` 标签上的 `class="card-image"`
- 添加了 `alt` 属性（提高可访问性和 SEO）

---

### 2. school.css 的修改

**修改前：**
```css
.card-image {
    width: 100%;
    height: 180px;
    background-size: cover;
    background-position: center;
    display: flex;
    align-items: center;
    justify-content: center;
    font-size: 60px;
    overflow: hidden;
}
```

**修改后：**
```css
/* 图片容器 */
.card-image {
    width: 100%;
    height: 200px;
    overflow: hidden;
    background-color: #f0f0f0;
    display: flex;
    align-items: center;
    justify-content: center;
}

/* 图片本身 - 新增 */
.card-image img {
    width: 100%;
    height: 100%;
    object-fit: cover;          /* 保持比例，裁剪多余部分 */
    object-position: center;    /* 从中心裁剪 */
    display: block;             /* 移除 img 底部空白 */
}
```

**关键点：**
- 删除了 `background-size` 和 `background-position`（这些是给背景图片用的）
- 新增了 `.card-image img` 样式
- 使用 `object-fit: cover` 让图片填满容器并保持比例

## 为什么之前不能正常显示？

### 问题 1：HTML 结构错误
```php
❌ <img class="card-image">  <!-- img 不应该有这个 class -->
✓ <div class="card-image"><img></div>  <!-- class 应该在外层 div -->
```

### 问题 2：CSS 样式冲突
- `.card-image` 原本是为 `background-image` 设计的
- 直接用在 `<img>` 标签上会导致图片变形
- 需要单独的 `.card-image img` 样式

### 问题 3：缺少 Content-Type header
- `get_image.php` 只输出二进制数据
- 浏览器不知道这是图片
- 必须添加 `header("Content-Type: image/jpeg")`

## 文件使用说明

### 1. 替换文件

将下载的文件替换到你的项目中：

```
your_project/
├── school.php          ← 替换这个
├── CSS/
│   └── school.css      ← 替换这个
└── get_image.php       ← 替换或新建这个
```

### 2. 检查数据库

确保数据库中的图片数据存在：

```sql
SELECT sid, school_name, LENGTH(image) as image_size 
FROM school 
WHERE image IS NOT NULL;
```

应该看到每个学校的图片大小（如 1500000 字节）。

1. **测试单张图片：**

   - ✗ 如果显示乱码，检查 `get_image.php` 的 header 设置

2. **测试整个页面：**
   访问：`http://localhost/your_project/school.php`
   
   - ✓ 所有卡片应该正常显示图片
   - ✓ 图片应该完整显示，不变形

## 常见问题

### Q1: 图片还是变形/压扁？
**A:** 检查 CSS 文件中 `.card-image img` 的样式是否正确添加。

### Q2: 图片显示不出来（空白）？
**A:** 
1. 检查 `get_image.php` 是否有 `header("Content-Type: ...")` 
2. 在浏览器按 F12，查看 Network 标签，看看请求是否成功

### Q3: 所有卡片显示同一张图片？
**A:** 检查数据库中 `sid` 字段是否唯一，以及 SQL 查询是否正确。

## object-fit 属性说明

| 值 | 效果 | 适用场景 |
|---|---|---|
| `cover` | 保持比例，填满容器，可能裁剪 | **当前使用** - 类似图2效果 ✓ |
| `contain` | 保持比例，完整显示，可能有空白 | 必须看到完整图片 |
| `fill` | 拉伸填满，不保持比例 | 不推荐 ✗ |

如果你想完整显示图片（可能有空白），改成：
```css
.card-image img {
    object-fit: contain;  /* 改这里 */
}
```
